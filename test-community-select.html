MobileStartInspectionPage - Community Select Fix Verification

Issue Analysis

Original Problem: UI locks up when selecting a community from dropdown
Root Cause: Controlled/uncontrolled component conflict in React Select

Fix Applied

1. Added local state: const [localCommunity, setLocalCommunity] = useState(preSelectedCommunity || '')
2. Added sync effect: useEffect to sync localCommunity with preSelectedCommunity
3. Updated handler: handleCommunityChange updates both local and parent state
4. Fixed Select value: Now uses localCommunity instead of preSelectedCommunity
5. Fixed validation: isFormValid now uses localCommunity
6. Fixed submit: handleStartInspection now uses localCommunity
